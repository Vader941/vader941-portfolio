L’Oréal Beauty Routines

AI-powered routine builder and product advisor

L’Oréal Beauty Routines lets visitors build a skincare or beauty routine around real products and then chat with a personal beauty advisor powered by OpenAI. Users can filter products by category, select the ones they care about, and ask detailed questions about when and how to use everything together.

Under the hood, I wired a static GitHub Pages front end to a Cloudflare Worker that securely calls the OpenAI API. The app keeps track of selected products in localStorage, so people can come back later and pick up where they left off.

Tech highlights

  • HTML, modern CSS, and vanilla JavaScript for the UI and product grid.
  • OpenAI API access proxied through a Cloudflare Worker so my API key never lives in the browser.
  • Conversation history and product context passed to the model for smarter, more personalized responses.
  • Local persistence of selected products using localStorage so sessions feel continuous.

Space Explorer

Daily images and facts from NASA’s archive

Space Explorer pulls in NASA’s Astronomy Picture of the Day over a custom date range and presents it in a clean, readable gallery. Set your dates, then scroll through high-resolution imagery with the official explanations alongside. You can even check what NASA published on your birthday.

This project was my way of getting comfortable with third-party APIs, error handling, and building a UI that feels good whether you are on a laptop or a phone.

Tech highlights

  • Fetches imagery and metadata from the official NASA API.
  • Responsive layout that adapts from a single-column mobile view to a richer desktop layout.
  • Date-range selection with basic validation and friendly error messages.
  • Hosted on GitHub Pages, using a Cloudflare Worker where needed to keep keys safe.

Pipeline to Hope

Awareness-driven puzzle game for clean water

Pipeline to Hope is a timed pipe-connecting game built for Charity: Water. The goal is simple: connect the pipes and get clean water to the village before the clock runs out. Behind the scenes, there is grid logic, collision checking, and state management doing the work.

I built this as a prototype to explore how simple games can support education and awareness, and to stretch my JavaScript skills beyond basic forms and DOM manipulation.

Tech highlights

  • Custom game loop and board logic written in vanilla JavaScript.
  • Responsive layout so the game is playable on both desktop and mobile.
  • Configurable difficulty and timing so the experience can be tuned.

Mini Games Arcade

A playground of JavaScript mini-games

The Mini Games Arcade is exactly what it sounds like — a collection of quick games I built to keep my JavaScript skills sharp. It includes small experiences that focus on input handling, basic physics, randomization, and clean UI feedback.

This is my “sandbox” project: a safe place to try ideas, refactor code, and practice writing logic that is easy to reason about when things get fast and chaotic on screen.

Tech highlights

  • Multiple self-contained JavaScript games under one consistent UI.
  • Reusable components for buttons, score panels, and layout.
  • Responsive design tweaks for comfortable play on laptops and tablets.
